# ARP

ARP 기본
---
> ARP(Address Resolution Protocol) 프로토콜이란<br>
```
OSI 7 Layer에서 Layer 3에 위치한 프로토콜이며, MAC 주소와 IP 주소를 서로 연결하는 용도로 사용한다.
LAN 환경에서는 MAC 주소를 기반으로 통신하기 때문에 ARP는 IP 주소를 기반으로 MAC 주소를 알아오는 역할
```
